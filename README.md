# An-Efficient-Chaos-based-hybrid-encryption-techniques-using-AI-and-FPGA

## Overview
Developed a hybrid encryption technique combining Chua's circuit and Lorenz system to generate pseudo-random sequences for secure image encryption.

## Features
- Hybrid chaotic encryption model
- Python-based simulation
- Security metric evaluation (UACI, NPCR, Entropy)
- Simulink hardware workflow design

## Technologies Used
- Python
- Simulink
- FPGA concepts
